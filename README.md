# 海南大成律所业务管理平台

基于Agent矩阵的智能化律所管理系统

## 技术栈

- 前端: React + Next.js
- 后端: Node.js + Python微服务
- 数据库: PostgreSQL + Elasticsearch
- 向量数据库: Weaviate
- 消息队列: RabbitMQ
- AI引擎: Claude API
- 权限引擎: Casbin

## 部署环境

- Windows Server 2022
- 腾讯云轻量服务器

## 目录结构

```
lawDC/
├── backend/          # Node.js后端服务
├── agents/           # Python Agent微服务
├── frontend/         # React前端
├── database/         # 数据库脚本
└── docs/            # 文档
```
